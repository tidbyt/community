# WoW Character Stats for Tidbyt
Show statistics for a World of Warcraft character. 

Stats shown include 
- Name 
- Class(icon)
- Character level
- Character faction
- Item level
- Mythic+ rating
- Raid progress

For testing via pixlet serve: http://127.0.0.1:8080/?region=&character=&realm=&client_id=&client_secret=
